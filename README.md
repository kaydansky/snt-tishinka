# СНТ Тишинка

## Описание проекта

"СНТ Тишинка" - это веб-приложение, которое предоставляет пользователям автоматическую консультацию по вопросам, связанным с садоводческим некоммерческим товариществом. Может использоваться для любых организаций и цифровых архивов. Приложение является клиентом, взаимодействующим с сервером по API. Сервер использует технологию RAG (Retrieval-Augmented Generation — генерация, дополненная поиском) для поиска информации в векторной базе данных и API для доступа к различным LLM. Сервер не является Open Source приложением и доступен только по запросу.

## Основные функции

- Интерактивный чат-бот для консультаций по вопросам СНТ
- Поддержка темной и светлой тем оформления
- Адаптивный дизайн для работы на различных устройствах
- Возможность установки как Progressive Web App (PWA)
- Поддержка многоязычности
- Защита от спама с помощью Cloudflare Turnstile
- Доступ к API методом CORS (Cross-Origin Resource Sharing)

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap Icons
- Cloudflare Turnstile для защиты от ботов
- Service Worker для PWA функциональности
- Web Share API для функции "Поделиться"

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/your-username/snt-tishinka.git
   ```

2. Перейдите в директорию проекта:
   ```
   cd snt-tishinka
   ```

3. Откройте файл `index.html` в вашем браузере или разместите файлы на веб-сервере.

## Использование

1. Откройте приложение в браузере
2. Введите ваш вопрос в поле ввода в нижней части экрана
3. Нажмите кнопку отправки или Enter для отправки вопроса
4. Дождитесь ответа от AI-ассистента

## Файловая структура

- `index.html` - Главная страница приложения
- `styles.css` - Стили приложения
- `app.js` - Основной скрипт приложения
- `chat-service.js` - Сервис для работы с AI-чатом
- `chat-widget.js` - Виджет чата
- `chat-init.js` - Инициализация чата
- `config.js` - Файл конфигурации
- `translations.js` - Файл переводов
- `turnstile.js` - Интеграция с Cloudflare Turnstile
- `sw.js` - Service Worker для PWA
- `manifest.json` - Манифест веб-приложения
- `assets/` - Директория с изображениями и favicon

## PWA функциональность

Приложение поддерживает установку как Progressive Web App. Для этого оно включает:
- Манифест веб-приложения (`manifest.json`)
- Service Worker (`sw.js`) для кэширования ресурсов
- Иконки для различных устройств
- Настройки для работы в автономном режиме
